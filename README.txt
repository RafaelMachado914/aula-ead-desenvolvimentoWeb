# 💡 Site criado para a aula de desenvolvimento web

## 📋 Descrição

Este projeto é um sistema simples feito apenas como teste para uma aula de desenvolvimento web usado no meu curso de graduação de ciencia da computação. para este projeto foi pego um template de um site e alterado para simular um site para o curso de ciência da computação. logo após finalizar o projeto os alunos deveriam subir o site no github

## ⚙️ Funcionalidades

✅ o projeto deve simular um site para o curso de ciencia da computação do intituto federal de Santa Catarina, para isto foi ultilizado um template do site https://html5up.net/ e alterado
para se adequar ao projeto
